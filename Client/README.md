# 🌐 SkyView Capital – Frontend

Frontend of **SkyView Capital**, a full-stack AI-powered stock analysis platform that provides users with insights and predictions for smarter trading decisions.

---

## 📌 Overview

This is the client-side application built using **React.js**, designed to provide:

- 📊 Interactive UI for stock analysis  
- 🔐 User authentication (Login/Signup)  
- 📈 Visualization of predictions  
- ⚡ Smooth and responsive user experience  

---

## ⚙️ Tech Stack

- React.js  
- HTML, CSS, JavaScript  
- Axios (API calls)  
- React Router  

---

## ✨ Features

- 🔐 Authentication (Login / Signup)
- 📊 Stock prediction interface
- 📈 Clean dashboard UI
- ⚡ Fast and responsive design
- 🔗 API integration with backend

---
